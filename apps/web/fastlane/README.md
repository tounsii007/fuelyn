# Fuelyn — Store metadata (fastlane)

Submission-ready App Store & Google Play listing metadata, in the layout
[`fastlane deliver`](https://docs.fastlane.tools/actions/deliver/) (iOS) and
[`fastlane supply`](https://docs.fastlane.tools/actions/supply/) (Android) read directly.

```
fastlane/metadata/
  <locale>/                 # iOS — App Store Connect (deliver)
    name.txt                # ≤ 30 chars
    subtitle.txt            # ≤ 30
    keywords.txt            # ≤ 100, comma-separated
    promotional_text.txt    # ≤ 170
    description.txt
  android/<locale>/         # Google Play (supply)
    title.txt               # ≤ 30
    short_description.txt    # ≤ 80
    full_description.txt     # ≤ 4000
```

Locales: **de-DE** (primary), **en-US**, **fr-FR**.

- Copy is regenerated by `scratchpad/gen-fastlane.js`, which also **validates the
  character limits** (App Store name/subtitle/keywords/promo, Play title/short).
- Positioning leads the effective-price differentiator; keywords also cover the
  generic incumbent terms (`spritpreise`, `tankstelle`, …) for discoverability.
- The annotated human overview lives in [`../store/store-listings.md`](../store/store-listings.md).
